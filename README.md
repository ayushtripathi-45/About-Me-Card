# 👤 Personal Self Card 

A clean, modern, and responsive **Personal Self Card** created using **HTML & CSS**.  
This project represents a personal profile card that highlights basic details, skills, and contact links in a structured layout.

--- 

## 📌 Overview

This **Self Card** project is designed to showcase personal and professional information in a compact card format.  
It can be used as:
- A GitHub profile showcase  
- A portfolio “About Me” card  
- A reusable personal profile component  

Built purely with **HTML and CSS**, it focuses on clean UI and readability.

---

## ✨ Features

- Responsive card layout  
- Clean and minimal UI  
- Well-structured information sections  
- Skill listing using semantic HTML  
- Easy to modify and reuse  
- No JavaScript used  

---

## 🛠️ Tech Stack

- **HTML5** – Page structure  
- **CSS3** – Styling and layout
